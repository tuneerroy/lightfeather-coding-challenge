# Lightfeather Full-Stack Engineer Coding Challenge

### Frontend: React 
- (port: 3000)
### Backend: Node.js (Express)
- (port: 8080)

## Instructions:
### Start: (website available at [localhost:3000](http://localhost:3000))
```bash
docker-compose build
docker-compose up
```
### Stop:
```bash
docker-compose down
```
or 
<kbd>Control</kbd> + <kbd>C</kbd>
